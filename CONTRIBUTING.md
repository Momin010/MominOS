# Contributing to MominOS Pro-X

We love contributions! Since this is a specialized OS project, please follow these guidelines:

## How to Contribute
1. Fork the repository.
2. Create a new branch for your feature or bugfix.
3. Keep assembly modules high-density and well-documented.
4. Verify your changes with the provided `scripts/build.bat`.
5. Submit a pull request.

## Coding Standards
- Use `nasm` syntax.
- Prefer modular driver development.
- Ensure 64-bit compatibility for kernel-level changes.
